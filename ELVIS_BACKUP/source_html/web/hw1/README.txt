This is the directory that stores all of the resources for Kevin Dittmar's Web Programming
Homework page.

Files named with the pattern .*shot.png are screenshots to demonstrate 
that the page works on:
	Internet Explorer
	Mozilla Firefox
	Google Chrome
	Opera
	Konqueror
	Lynx
	W3M

Files named with the pattern logo_*.png are logos for each of the 
graphical web browsers to be used as links to their respective testing
screenshots.

The picture black_mage.png is the icon for the webpage.

The picture photo_of_self.png is a picture of me to help Professor 
Provine place a face to a name when grading my work.

The picture HTML5_Logo.png is the official logo for HTML5 to be used 
as a link to the W3C HTML5 validator site that validates my webpage.

The picture vcss.gif is the official validation logo for CSS3 to be 
used as a link to the W3C CSS3 validator site that validates my webpage.

The HTML file hw1.html includes all of the HTML for this webpage.

The CSS file webline.css includes all of the CSS for the webline
on this webpage.

The CSS files layout699-.css and layout700+.css control the webpage's
responsive style changes.  The file layout699-.css is intended for 
tailor the display for devices with a small screen, like a smart phone
or other mobile devices, and the file layout700+.css changes the styling
to be appropriate for full screen devices, like a laptop or desktop 
computer.
